**The Cauchy-Schwarz Inequality**

$$
  \sum_{k=1}^n a_k b_k
  \leq 
  \Big( \sum_{k=1}^n a_k^2 \Big)^{1/2} \Big( \sum_{k=1}^n b_k^2 \Big)^{1/2}
$$
